# 🤖 AI Doubt Solver

An AI-powered chatbot that answers questions in **short, simple, and easy-to-understand points**.  
Built using **Next.js** and **Google Gemini API**.

---

## ✨ Features

- 💬 Ask any question
- ⚡ Instant AI answers
- 📌 Short point-wise responses
- 🎨 Smooth message fade animation
- 🧠 Powered by Gemini AI

---

## 🛠️ Tech Stack

- ⚛️ Next.js
- 🎨 CSS
- 🤖 Gemini API
- ☁️ Vercel (for deployment)

---

## 🚀 How to Run Locally

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Run the project

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```

---

## 🌐 Deployment

The project can be easily deployed using **Vercel**.

Just connect your GitHub repository and add the environment variable:

```
GEMINI_API_KEY
```
